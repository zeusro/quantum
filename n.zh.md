# n维世界量子运算法则

第一性原理：**n=...=4=3=2=1=0** && a²=1

符号定义：

= ：表示一种基于时间的运算结果。计算过程是单向不可逆的。

1 ： 1个量子。

## 基本加法

### 1+1=2

$$
1+1=1+1 ∧ a²=1 ⟹ 1+a=1+a ∧ a=0 ∧ a=1 ⟹ 1+1=1+0 ∧ 1=2 ⟹ 1+1=2
$$

### 1+1=3

$$
2=3 ∧ 1+1=2 ⟹ 1+1=3
$$

### 3-1=1

$$
1+1=3 ⟹ 3-1=1
$$

### 1-1=3

$$
3-1=1 ∧ 3=1 ⟹ 1-1=3
$$

### 1(+/-)1=n

$$
1+1=3 ∧ 3 = n ∧ 1-1=3 ⟹ 1(+/-)1=n
$$

## 量子不等式

### n < n + 1

$$
n=n ∧ 0 < 1 ⟹ n < n + 1
$$

### n < n - 1

$$
1+1=3 ∧ 1-1=3 ⟹ +=- ∧  n < n +1 ⟹ n < n - 1
$$

### n < n +/- 1

$$
n < n + 1 ∧ n < n - 1 ∧ +=- ⟹ n < n +/- 1
$$
