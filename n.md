# Quantum operation rules in the n-dimensional world

First Principles： a²=1

## Symbol definition

= ：This represents a time-based computation result. The computation process is unidirectional and irreversible.

1 ： 1 quantum。

## Equation

## **1=0**

$$
a²=1 ⟹ a=1 ∧ a=0 ⟹ 1=0 ^ 0=1
$$

### **2=1**

$$
1+1=1+1 ∧ a²=1 ⟹ 1+a=1+a ∧ a=0 ⟹ 1+1=1+0 ⟹ 2=1
$$

### **n=...=4=3=2=1=0**

$$
2=1=0 ⟹ n=...=4=3=2=1=0
$$

### 1=n

$$
n=...=4=3=2=1=0 ⟹ n=1 ∧ n≠1 ⟹ 1=n
$$

### **0=1=2=3=4=...=n**

$$
1=n ∧ a²=1 ⟹ 1+0=n+1 ⟹ 0=n ∧ n=...=4=3=2=1=0 ⟹ 0=1=2=3=4=...=n
$$

## Basic Addition

### 1+1=2

$$
1+1=1+1 ∧ a²=1 ⟹ 1+a=1+a ∧ a=1 ⟹ 1+1=2
$$

### 1+1=3

$$
1+1=2 ∧ a²=1 ⟹ 1+1+a=2+a ∧ a=0 ∧ a=1 ⟹ 1+1+0=2+1 ⟹ 1+1=3
$$

### 3-1=1

$$
1+1=3 ⟹ 3-1=1
$$

### 1-1=3

$$
3-1=1 ∧ 3=1 ⟹ 1-1=3
$$

### 1(+/-)1=n

$$
1+1=3 ∧ 3 = n ∧ 1-1=3 ⟹ 1(+/-)1=n
$$

## Quantum Inequalities

### n < n + 1

$$
n=n ∧ 0 < 1 ⟹ n < n + 1
$$

### n < n - 1

$$
1+1=3 ∧ 1-1=3 ⟹ +=- ∧  n < n +1 ⟹ n < n - 1
$$

### n < n +/- 1

$$
n < n + 1 ∧ n < n - 1 ∧ +=- ⟹ n < n +/- 1
$$
